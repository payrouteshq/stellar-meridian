# Meridian

> **Status: proposal.** Meridian does not exist yet. This is what we are proposing to build for the Stellar ecosystem. Nothing here is installable. If you're interested in seeing this built, watch the repo.

**The headless tokenization framework for Stellar.**

issue, manage and transfer regulated assets on Stellar without touching a vendor UI. composable primitives. agent-ready. you own the code.

```bash
# not yet available
npx shadcn add @meridian/asset-issuer
```

## What's inside

| Primitive | Package |
|---|---|
| TypeScript SDK | `@meridian/stellar-core` |
| Soroban Contracts | transfer restriction + distribution |
| shadcn Registry | `npx shadcn add @meridian/...` |
| MCP Server | `meridian-mcp` |
